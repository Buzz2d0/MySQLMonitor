## MySQLMonitor
> 监控 MySQL 查询、执行语句...

**Usage:**
```bash
$ ./bin/MySQLMonitor 
Usage: MySQLMonitor [options]

  -h    Shows usage options.
  -host string
        Bind mysql host. (default "localhost")
  -port uint
        Bind mysql port. (default 3306)
  -user string
        Select mysql username.
  -passwd string
        Input mysql password.
```
